# **App Name**: HostelDesk

## Core Features:

- Public Landing Page: Display a landing page with app info and a 'Sign in with Google' button.
- User Dashboard: Display a dashboard showing a list of hostels created by the logged-in user.
- Hostel Creation: Allow users to add new hostels via a form/modal.
- Hostel Data: Read data of each hostel based on the userID to manage.

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) - A clean blue (#478DE0) for a trustworthy feel.
- Background color: HSL(210, 20%, 98%) - Very light desaturated blue (#FAFCFF) for a calm backdrop.
- Accent color: HSL(240, 50%, 50%) - A vibrant violet (#5A5EE0) for highlighting interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Clean, modern, and responsive layout with intuitive navigation.
- Use clear and modern icons to represent actions and data points.
- Subtle animations for transitions and feedback on user interactions.
